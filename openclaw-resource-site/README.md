# OpenClaw 资源站

一个开箱即用的开源 AI 助理教程与资源网站。

## 在线演示

部署后可访问（如部署到 GitHub Pages 或 Vercel）。

## 部署方式

### 方式一：GitHub Pages（免费 ✅ 推荐）

1. 在 GitHub 创建新仓库
2. 把 `openclaw-resource-site/` 文件夹里的内容推上去
3. 仓库 Settings → Pages → 选择 main 分支 → 保存
4. 几分钟后访问 `https://你的用户名.github.io/仓库名/`

### 方式二：Vercel（免费，一键部署）

```bash
npx vercel
```

### 方式三：本地预览

直接用浏览器打开 `index.html` 即可。

## 盈利模式

- **Google AdSense**：在 HTML 中已预留广告位（查找 `adsbygoogle` 注释）
- **联盟营销**：推荐 VPS、域名、AI 服务等赚取佣金
- **付费模板**：资源下载页可放置付费配置模板
