🔒 隐私政策

最后更新：2026年5月10日

信息收集

本网站使用 Google AdSense 展示广告，Google 可能会使用 Cookie 来展示基于您兴趣的广告。本网站本身不收集、存储或处理任何个人身份信息。

如果您通过本网站下载资源或提交评论，我们仅收集您主动提供的信息（如电子邮件地址），仅用于回复您的咨询。

Cookie 使用

本网站使用的第三方服务（如 Google AdSense）可能在其服务范围内设置 Cookie，用于提供个性化广告体验。您可以通过浏览器设置管理或删除 Cookie。

第三方服务

本网站使用以下第三方服务：

  • Google AdSense — 展示广告，可能使用 Cookie 和网络信标
  • GitHub Pages — 网站托管服务

这些第三方服务有各自独立的隐私政策，我们不对其内容或行为负责。

数据安全

我们采用合理的技术措施保护网站安全，但由于互联网的开放性，无法保证数据传输的绝对安全。

联系我们

如有关于本隐私政策的疑问，请通过 GitHub Issues 联系我们。

政策更新

我们可能不时更新本隐私政策。更新后的政策将在本页面发布，并注明最后更新日期。
