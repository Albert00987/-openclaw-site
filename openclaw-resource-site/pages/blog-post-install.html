📥 OpenClaw 零基础安装教程

Windows · Mac · Linux 全平台覆盖 | 2026年5月10日

还在用网页版 AI 聊天？试试把自己的 AI 助理装在电脑上——数据归你、24小时在线、还能接入微信和QQ。

OpenClaw 是一个开源的个人 AI 系统。这篇教程带你从零开始，5 分钟装好。

💡 你需要：一台能上网的电脑 + 5 分钟时间。不需要编程基础。

第一步：安装 Node.js

OpenClaw 基于 Node.js 运行。打开 nodejs.org，下载最新 LTS 版本（推荐大多数人用的稳定版）。

Windows 用户

下载 .msi 安装包 → 双击运行 → 一路点"Next" → 装完重启终端（CMD 或 PowerShell）。

Mac 用户

下载 .pkg 安装包 → 双击安装。或者用 Homebrew：

brew install node

Linux 用户（Ubuntu/Debian）

curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs

验证安装

打开终端，运行以下命令，看到版本号就说明装好了：

node --version
npm --version

第二步：安装 OpenClaw

在终端输入一行命令：

npm install -g openclaw

等进度条跑完（约 30 秒到 1 分钟，取决于你的网速）。

⚠️ Mac/Linux 如果遇到权限错误，加 sudo：sudo npm install -g openclaw

第三步：启动

openclaw dev

首次启动会自动初始化配置，稍等几秒。

第四步：打开 Web 界面

打开浏览器，访问：

http://localhost:18789

你会看到一个漂亮的设置向导。

第五步：配置 AI 模型

在向导中选择一个 AI 模型。新手推荐 DeepSeek：

  1. 在模型列表中选择 DeepSeek
  2. 去 platform.deepseek.com 注册账号，获取 API Key
  3. 把 API Key 粘贴到配置页面 → 保存
💡 DeepSeek 是目前性价比最高的选择。注册送 500 万 token，中文能力强，价格低。够个人用好几个月。

🎉 搞定了！

现在你的 AI 助理已经开始运行了。继续配置以下功能，让它变得更强：

  • 接入 QQ Bot — 用手机 QQ 随时指挥你的 AI
  • 接入微信 — 扫码登录，微信上直接对话
  • 设置心跳巡检 — AI 每 30 分钟自动检查消息、待办、日程
  • 自定义人格 — 修改配置文件，让 AI 拥有专属性格

❓ 常见问题

Q：端口被占用怎么办？

启动时指定其他端口：openclaw dev --port 18888

Q：怎么修改 AI 的性格？

编辑 ~/.openclaw/workspace/SOUL.md 文件，写入你想要的风格即可。

Q：安装报错了怎么办？

最常见的原因是网络问题或权限问题。尝试：

  • Mac/Linux：加 sudo 重试
  • Windows：以管理员身份运行终端
  • 切换网络环境（比如手机热点）再试
