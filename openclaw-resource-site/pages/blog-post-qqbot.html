📱 如何将 OpenClaw 接入 QQ Bot

手机 QQ 随时指挥你的 AI 助理 | 2026年5月10日

装好了 OpenClaw，但每次都要打开浏览器才能用？把它接上 QQ Bot，以后在手机上发个消息就能指挥 AI。

💡 这是最推荐的远程使用方式。QQ 的机器人接口稳定、免费，而且不需要企业资质。

前提条件

  • ✅ OpenClaw 已安装并正常运行
  • ✅ 一个 QQ 账号（个人号即可）
  • ✅ 10 分钟时间

第一步：注册 QQ 机器人

  1. 打开 QQ 开放平台
  2. 用你的 QQ 号登录
  3. 点击"创建机器人"
  4. 填写基本信息（机器人名称、头像、介绍）
  5. 创建成功后，进入"开发设置"页面
  6. 记下 AppID 和 Client Secret（不要泄露！）
⚠️ 机器人创建后需要审核，通常 1-2 个工作日。但你可以先配置好，审核通过直接使用。

第二步：配置 OpenClaw

在终端中依次执行以下命令：

openclaw config set channels.qqbot.appId "你的AppID"
openclaw config set channels.qqbot.clientSecret "你的ClientSecret"
openclaw config set channels.qqbot.allowFrom '["*"]'

第三步：重启 Gateway

openclaw gateway restart

第四步：添加机器人为好友

在 QQ 中搜索你的机器人（用 AppID 或机器人名称），加为好友。发送任意消息，就能和 AI 对话了。

💡 如果发送消息后没有回复，检查终端日志：openclaw logs --tail 20

🎉 完成！

现在你可以在 QQ 上随时指挥你的 AI 助理了。试试发一条"帮我查一下今天的天气"或"整理一下我的待办事项"。
