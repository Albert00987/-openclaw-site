📥 OpenClaw 安装教程

从零开始，手把手装好你的 AI 助理

📋 安装前准备

你只需要：一台能上网的电脑（Windows / Mac / Linux 都可以），以及 5 分钟时间。

💡 最低配置：2GB 内存，500MB 磁盘空间即可。OpenClaw 本身很轻量。

Step 1：安装 Node.js

OpenClaw 基于 Node.js 运行，所以第一步先装它。

👉 打开 nodejs.org，下载最新版本的 LTS（长期支持版）

各系统安装方式：

Windows

下载 .msi 安装包 → 双击运行 → 一路"下一步"装完即可。完成后重启终端（CMD 或 PowerShell）。

Mac

下载 .pkg 安装包 → 双击运行 → 一路安装完成。或通过 Homebrew：

brew install node

Linux（Ubuntu/Debian）

curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs

验证安装

打开终端，输入以下命令，看到版本号就说明装好了：

node --version
npm --version

Step 2：安装 OpenClaw

一行命令搞定安装：

npm install -g openclaw

等待进度条跑完（约 30 秒 - 1 分钟，取决于网速）。

💡 Mac/Linux 用户如果遇到权限错误，加 sudo： 
sudo npm install -g openclaw

Step 3：启动 OpenClaw

安装完成后，运行：

openclaw dev

首次启动会自动初始化配置。

Step 4：打开 Web 界面

打开浏览器，访问：

http://localhost:18789

你会看到一个设置向导界面。

Step 5：配置 AI 模型

在向导中选择一个 AI 模型。推荐新手用 DeepSeek：

  1. 在模型列表中选择 DeepSeek
  2. 注册 DeepSeek 账号 获取 API Key
  3. 把 API Key 填入配置页面
  4. 点击保存，搞定！
💡 省钱窍门：DeepSeek 价格最便宜，中文能力也好。注册送 500 万 token，够用很久。

🎯 进阶：接入 QQ Bot

配好之后，你在 QQ 上就能直接和 AI 聊天了。

步骤：

  1. 去 QQ 开放平台 注册一个机器人应用
  2. 获取 appId 和 clientSecret
  3. 在终端运行：
openclaw config set channels.qqbot.appId "你的appId"
openclaw config set channels.qqbot.clientSecret "你的Secret"
openclaw config set channels.qqbot.allowFrom '["*"]'
openclaw gateway restart

🎯 进阶：接入微信

扫码登录，微信上就能使唤 AI：

openclaw plugins install "@tencent-weixin/openclaw-weixin"
openclaw config set plugins.entries.openclaw-weixin.enabled true
openclaw gateway restart
openclaw channels login --channel openclaw-weixin

终端会弹出二维码，用微信扫码确认即可。

❓ 常见问题

Q：安装提示权限错误？

Mac/Linux 用 sudo 前缀运行安装命令。Windows 以管理员身份打开终端。

Q：端口 18789 被占用了？

启动时指定其他端口：

openclaw dev --port 18888

Q：怎么改 AI 的性格？

编辑 ~/.openclaw/workspace/SOUL.md 文件，写下你想要它具备的人格特征。

⚠️ 注意：OpenClaw 目前主要是终端/Web 界面，没有原生图形桌面客户端。但它可以通过 QQ、微信、Telegram 等聊天平台和你交互。
