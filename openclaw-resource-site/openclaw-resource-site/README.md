## 🚀 这是什么？

**OpenClaw** 是一个开源的个人 AI 系统，运行在你的电脑上，数据完全属于你。

它可以：
- 🔗 接入微信、QQ、企业微信、Telegram、Discord
- 🧠 支持 DeepSeek、Claude、GPT 等数十种 AI 模型
- ⏰ 定时巡检，自动查看消息、待办、日程
- 🛠️ 自定义人格，打造你的专属 AI 助理
- 🔒 数据本地化，隐私安全无忧

本资源站提供 OpenClaw 的：
- ✅ **安装教程**：Windows / Mac / Linux 手把手教学
- ✅ **渠道接入指南**：QQ Bot、微信、企业微信等
- ✅ **配置文件模板**：开箱即用的配置方案
- ✅ **AI 人格预设**：外贸业务员、文案写手、学习助手等
- ✅ **实战案例**：市场调研、开发信撰写等

## 🌐 部署方式

### GitHub Pages（推荐 · 免费）

1. 将本仓库 Settings → Pages → Source 设为 main 分支
2. 等待 2 分钟，访问 `https://你的用户名.github.io/openclaw-site/`

### Vercel（免费 · 一键部署）
